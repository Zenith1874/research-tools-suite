# 美国宏观五支柱扩展状态（2026-07-17）

## 目标

把 `/us-macro` 扩展为“就业—通胀—增长—财政债务—金融条件”五支柱页面，同时保持来源、频率、发布日期和派生口径可审计。

## 新增原始系列

- 就业：U-3/U-6、参与率、就业人口比、非农、首次申领、JOLTS 空缺/招聘/裁员/离职、平均时薪与劳动生产率。
- 通胀：`CPIAUCSL` CPI、`CPILFESL` 核心 CPI、`PCEPILFE` 核心 PCE。
- 增长：`GDPC1` 实际 GDP、`INDPRO` 工业产出、`RSAFS` 零售和餐饮销售。
- 财政债务：BEA 联邦经常性收入/支出、财政年度盈余/赤字、公共债务总额与 GDP 占比、联邦利息支付。
- 金融条件：联邦基金、SOFR、优惠贷款利率、3M/2Y/5Y/10Y/30Y 美债、10Y-2Y 利差、10 年盈亏平衡通胀和 30 年房贷。

原始 observation 全部保存为 `official`，来源页为相应 FRED series 页面，本源写入 `parser_notes`。单系列失败不清除其他系列或旧数据。
更新和读取入口显式关闭 SQLite 连接，避免长期运行积累文件句柄或在 Windows 下锁住临时数据库。

## 派生指标

- 非农就业月增 = `PAYEMS(t) - PAYEMS(t-1)`。
- CPI / 核心 CPI / 核心 PCE 同比 = `(index(t) / index(t-12) - 1) × 100`。
- 实际 GDP 环比年化 = `((GDPC1(t) / GDPC1(t-1))^4 - 1) × 100`。
- 工业产出与零售同比使用同一 12 个月变化公式。
- 平均时薪同比和劳动生产率同比分别使用 12 个月、4 个季度变化。
- 联邦经常性收支差额 = 同期 `FGRECPT - FGEXPND`；只匹配同一季度。

派生指标只在 API 响应中生成并标记 `derived`，不写入 `us_macro_observations`，不会与官方原始指数混淆。

## 页面结构

- 五个主题区块各自显示关键卡片和趋势图。
- 图表提供 5 年、10 年、全部历史窗口，默认 10 年，避免百年序列压缩近期变化。
- 劳动参与率、期限利差和房贷利率分别绘图，避免不同量级或不同经济含义相互压缩。
- 每张卡片显示自己的最新期，避免把不同发布日期的指标假装成同一个月份。
- 原始值与派生值使用 `official` / `derived` 标签和公式区分。
- 日频长序列仅在响应图表层降采样，数据库保存完整历史。

## 数据缺失与时间对齐复核

- 根因不是数据库大面积缺数，而是旧图表将各系列独立降采样后做日期并集，并在其他系列无当日值时错误断线；横轴还按记录序号而非真实日期定位。
- 已改为按真实时间戳定位横轴；单个系列在联合日期网格中的空值不再切断该系列；鼠标位置也按真实时间寻找最近 observation。
- 月度和季度序列逐期审计：除 `UNRATE`、`U6RATE`、`CIVPART`、`EMRATIO`、`CPIAUCSL`、`CPILFESL` 的 2025-10 在 FRED 原始 CSV 中共同缺失外，其余接入的月度/季度序列没有内部缺期。该期不插值、不伪造。
- 修复后重新完成桌面真实浏览器检查：政策利率三条日频曲线与五期限美债曲线均连续显示并按日期对齐，控制台 0 错误。
